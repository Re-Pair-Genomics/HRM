# Frontend source code
This directory contains the source code for the frontend of the application.
```
assets/ - Contains all the static assets like images, fonts, etc.
components/ - Contains all the reusable React components.
contexts/ - Contains all the contexts for managing global state with React Context API.
hooks/ - Contains all the custom hooks.
pages/ - Contains all the pages of the application. ???
routes/ - Contains all routing-related components and setup.
store/ - Contains the Redux store configuration and reducers. ??? If we use Redux
utils/ - Contains all the reusable utility functions.
services/ - Contains all the services for API calls to the backend.
App.js - The main React application component.
index.js - The entry point for the React application.

```
